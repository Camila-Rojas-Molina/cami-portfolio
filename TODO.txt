DAY 4 — May 24 (Sun) | THE UNIQUE FEATURE — LIVE ML DEMO
====================================================================
This is what makes your portfolio different. The idea:
an interactive ML demo that runs entirely in the browser using
TensorFlow.js — no backend needed, no API calls.

Suggested demo (pick one based on your strongest project area):

  OPTION A — Real-time object detection (CV focus)
    - Use TensorFlow.js + COCO-SSD or MobileNet
    - User enables webcam → model detects objects live
    - Label bounding boxes with confidence scores
    - Install: npm install @tensorflow/tfjs @tensorflow-models/coco-ssd

  OPTION B — Sentiment analysis (NLP focus)
    - User types a sentence → model predicts sentiment in real time
    - Use a small distilled BERT or Universal Sentence Encoder
    - Visualize confidence scores as animated bars
    - Install: npm install @tensorflow/tfjs @tensorflow-models/universal-sentence-encoder

  OPTION C — Neural network visualizer (general AI)
    - Animated, interactive diagram of a neural network
    - Inputs flow through layers with animated "activations"
    - No model needed — pure canvas/SVG animation
    - Works great with Framer Motion

[ ] Pick your option and install required packages
[ ] Build the Demo section with a brief explanation of what it does
[ ] Add loading state while model initializes
[ ] Make it mobile-friendly (graceful fallback if no webcam)
